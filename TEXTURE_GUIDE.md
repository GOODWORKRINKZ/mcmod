# Текстура Демогоргона
# Это текстовое представление. В реальной игре нужен PNG файл 64x64

Цветовая схема:
- Тело: Темно-серый (#3d1f1f) - цвет кожи
- Лепестки головы: Красновато-коричневый (#8b0000) - цвет мяса
- Внутренняя часть лепестков: Розовый (#ff69b4) - цвет слизистой
- Руки/ноги: Темно-серый с текстурой

Для создания настоящей текстуры:
1. Откройте любой графический редактор (Paint.NET, GIMP, Photoshop)
2. Создайте изображение 64x64 пикселей
3. Используйте эту раскладку:
   - Верх (0-16): Голова и лепестки
   - Средина (16-32): Тело
   - Низ (32-64): Руки и ноги
4. Сохраните как demogorgon.png
5. Положите в: src/main/resources/assets/strangerthingsmod/textures/entity/

Пока мод будет использовать текстуру по умолчанию (розовый/черный квадрат).
