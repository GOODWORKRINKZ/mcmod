# Stranger Things Mod для Minecraft 1.20.1

## Описание
Этот мод добавляет в Minecraft элементы из сериала "Очень Странные Дела" (Stranger Things):

### Возможности мода:
- **Демогоргон** - опасный враждебный моб с уникальными способностями
- **Изнанка** - новое измерение с темной атмосферой
- **Порталы** - Демогоргон может открывать порталы в Изнанку и затягивать игроков

## Характеристики Демогоргона:
- **Здоровье**: 60 HP (30 сердец)
- **Урон**: 8 HP (4 сердца)
- **Броня**: 4
- **Скорость**: Быстрее обычных мобов
- **Способности**:
  - Может открывать порталы в Изнанку (появляется портал 3x3)
  - Затягивает ближайших игроков к порталу
  - Агрессивен к игрокам
  - Спавнится в Изнанке

## Изнанка (The Upside Down):
- Темное измерение с постоянной ночью
- Мрачная атмосфера с темными цветами
- Высокая частота спавна монстров
- Демогоргоны естественно спавнятся здесь

## Установка:
1. Установите Minecraft Forge 1.20.1 (версия 47.2.0 или выше)
2. Скомпилируйте мод командой: `gradlew build`
3. Поместите скомпилированный .jar файл в папку mods
4. Запустите Minecraft

## Разработка:
Для разработки используйте:
```bash
# Запуск клиента
gradlew runClient

# Запуск сервера
gradlew runServer

# Сборка мода
gradlew build
```

## Команды для доступа в Изнанку:
```
/execute in strangerthingsmod:upside_down run tp @s ~ ~ ~
```
