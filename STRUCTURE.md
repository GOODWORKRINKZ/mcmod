# Stranger Things Mod - Структура проекта

## Основные компоненты:

### 1. Главный класс мода
- `StrangerThingsMod.java` - точка входа, регистрация всех компонентов

### 2. Сущности (Entities)
- `ModEntities.java` - регистрация сущностей
- `DemogorgonEntity.java` - логика моба Демогоргона
  - AI для атаки игроков
  - Способность открывать порталы
  - Механика затягивания игроков
- `DemogorgonRenderer.java` - рендеринг моба

### 3. Блоки (Blocks)
- `ModBlocks.java` - регистрация блоков
- `UpsideDownPortalBlock.java` - портальный блок
  - Телепортация между измерениями
  - Частицы и звуки портала
- `UpsideDownTeleporter.java` - логика телепортации

### 4. Измерения (Dimensions)
- `ModDimensions.java` - регистрация измерения
- `upside_down.json` - конфигурация измерения
- `upside_down_biome.json` - биом Изнанки

### 5. События (Events)
- `ModEventBusEvents.java` - обработка событий мода

## Файлы конфигурации:

- `build.gradle` - конфигурация сборки
- `mods.toml` - метаданные мода
- `settings.gradle` - настройки Gradle

## Ресурсы:

### Модели и текстуры:
- `models/block/` - модели блоков
- `models/item/` - модели предметов
- `blockstates/` - состояния блоков

- `lang/ru_ru.json` - русский
